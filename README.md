# 🍉 GuaDAO / 吃瓜群众自治社

> 你不只是看客，你是未来的共建者。

GUA DAO（吃瓜群众自治社）是一个由观众共建的内容自治社区：  
用 **贡献积分（GUA Points）** 连接 **提案、投票、共创与奖励**，并逐步升级到链上治理与代币化（GUA Token）。

---

## 🔥 这是什么？

- 🎥 **内容共创**：由社区投票决定选题、内容方向、活动玩法  
- 🍉 **积分系统**：互动/贡献获得 GUA Points，未来可映射/兑换 GUA Token（规划中）  
- 🗳️ **治理机制**：从“轻量投票”逐步过渡到“链上治理”  
- 🏆 **荣誉体系**：OG 徽章 / NFT（规划中），记录早期贡献与权益

---

## ❤️ 我们的价值观

- 🎯 自治而非独裁：方向由群众共决  
- 🎁 贡献即奖励：每一次互动、每一份贡献都值得被记录  
- 📚 学习即成长：当前阶段以内容教育与认知升级为主  
- 🤝 共建即共享：影响力与价值由早期成员共同拥有

---

## 🧭 Roadmap

- **v0.1（现在）**：白皮书 + 规则草案 + 协作流程 + 积分表/投票表  
- **v0.2**：可运行的积分服务 / 任务系统 / 投票系统（轻量）  
- **v0.3**：链上方案评估（Snapshot / Aragon / 自建合约）+ 数据公开透明  
- **v1.0**：GUA Token 上链与治理模块（视社区共识与合规情况推进）

> 注：所有关键升级以社区提案与投票结果为准。
